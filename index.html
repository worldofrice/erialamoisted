Ingliskeelne mõiste Eestikeelne vaste Eestikeelne seletus
Charset Märgisüsteem Kodeering, mille järgi arvuti kirjutab välja tähe, nummbri või millegi muu
Inline element Joondatud element Loob antud elemendile kujutletava joone, mida mööda kõik muud elemendid joonduvad
Block element Bloki element Iga uus element algab uuel real ning nendele on määratud automaatselt vahed
Padding Elemendisisene vahe Loob elemendi sisu ja elemendi äärte vahele määratud ruumi
Margin Elemendiväline vahe Loob elemendi äärte ja ekraani vahele määratud ruumi
Shorthand Lühend Saad mitme erineva omaduse asemel määrata elemendile ühe pikema omaduse
Wireframe Lehekülje mustand Kerge mustand lehekülje välimusest ja vajalike asjade paigutusest. Pole interaktiivne
Media query Meedia päring Nn. tehnika, mille puhul saab kasutada konditsionaalsust, et kasutada teatud omadusi ainult siis, kui miski vastab millelegi
Hover Hõljuma (millegi peal) Hiirekursoriga mingi elemendi (nt. nupu) kohal olema
Border-Box Piiri kast Lühend, mis loob elemendi ümber nähtava äärise
Padding-Box Kastisisene vahe Määra vahe kasti sisust äärteni
Scroll Kerima Mingil dokumedil või leheküljel kerima, et näha sisu, mis on ekraani kuvasuurusest väljas
Content stacking Sisu virnastamine Elementide järjekorra märkimine ja seetõttu ka nende kuhjamine üksteise otsa. Kõige esil on 1 ja sellest tagapool on sellest madalamad numbrid
Hypertext Hüperlink Tekst, mis võimaldab teistesse punktidesse või teistele materjalidele viidata, sellele vajutades
Hyperweb Hüperveeb iOS'i lisand, mis teeb netis surfamise mugavamaks ja kiiremaks
Viewport Vaatekuva Ekraanil vaadeldav pikslite ala
Semantic markup Semantiline ülesmärge Elemendi eraldi kujundamise asemel on kasutusel kasutajaliidese terminid
Checkbox Märkeruut Kast, mida saad valida hiireag peale vajutades
Radio buttons Raadio nupud Sarnane märkeruuduga. Erinevuseks on see, et raadio nuppe saab valida mitu tükki korraga
Dropdown select Rippmenüü Nupp, millele peale vajutades avaneb menüü erinevaid muid valikuid,mida saad vajutada
Input Sisend Mingisugune tegu, mis saadab arvutile käskluse midagi teha (kas hiireklõps või päring arvutile) või seadme küljes olev pesa, kust see saab infot või energiat
Disabled Välja lülitatud Mingi teenuse või funktsiooni välja lülitamine
Tooltip Tööriistavihje Hiirekursori juurde tekkiv tekst, mis seletab lahti, mida antud nupp (vms.) teeb
Toggle switch Lüliti Lüliti, mida saab sisse ja/või välja lülitada
Autocomplete Automaatne lõpetaja Funktsioon, mis proovib kasutaja eelnevalt kasutatud sõnade, konteksti või harjumuste järgi lõpetada kirjutatut. Proovib ennustada, mida tahad kirjutada
Slider Juhtriba Enamjaolt mingi riba, millel hiirega klõpsates, all hoides ja samal ajal libistades saab millegi väärtust muuta (nt. ekraaniheledust)
Root element Juurelement Hõlmab kõiki teisi elemente ning on lähteelement
Grid layout Ruudustikupaigutus On võimalus elemente kujutada arvutustabeli formaadis (nagu MS Excel)
UI library Kasutajaliidese sõnastik Sõnastik, kus on kasutusel kasutajaliidese terminid
Sidebar Külgriba Enamasti ekraani küljel olev riba, mille abil saab kas navigeerida mööda lehekülge kiiremini või siis kasutada erinevaid tööriistu (nt. MS Teams'i vasakul küljel)
